ScrollWatch

Easily add lazy loading, infinite scrolling, or any other dynamic interaction based on scroll position.

Download

Features

  • Watch any scrolling element, not just the window
  • Supports multiple instances and nested containers
  • Executes callbacks when elements enter/leave view
  • Tracks the first time something enters/leaves view or every time
  • Watches vertical and horizontal scrolling
  • Supports infinite scrolling out of the box
  • UMD wrapper for use in AMD, CommonJS or global environment
  • Scroll/Resize handlers are throttled, so your cpu lives to see tomorrow
  • No dependencies
  • Modern browser support and IE10+ (IE9+ with classList.js polyfill)
  • insert size

Basic Usage

Include the library on your page.

<script src="ScrollWatch.js"></script>

Create a new ScrollWatch instance and pass in an options object with an onElementInView callback. By default, ScrollWatch will fire your callback when any elements with a data-scroll-watch attribute come into view. Check out the API section below for all options. 

var sw = new ScrollWatch({
	onElementInView: function(data) {
		console.log(data.el, '...is now in view');
	}
});

API

Options

Below are all of the options you can pass to ScrollWatch and their default values:

{
	// The element that contains the elements you want to watch.
	// Remember, you can have multiple instances and watch nested containers.
	container: window.document.documentElement,

	// A selector for the element(s) you want to watch.
	watch: '[data-scroll-watch]',

	// True to stop watching the element after it comes into view the first time.
	// False to watch the element forever.
	watchOnce: true,

	// Class applied to elements that are in view.
	inViewClass: 'scroll-watch-in-view',

	// Number of milliseconds to wait between processing scroll events.
	scrollThrottle: 250,

	// Number of milliseconds to wait between processing resize events.
	resizeThrottle: 250,

	// Number of pixels to offset the viewable area when processing elements.
	// For example, if a watched element is 5 pixels below the fold, it will
	// be considered in view if watchOffset is > 5 pixels.
	watchOffset: 0,

	// Callback for when an element comes into view. A data object will be
	// passed into the callback, consisting of the following:
		// eventType: scroll, resize, refresh or scrollwatchinit if the element
		// is in view on initial load of the page
		// el: the element being watched
		// direction: up, down, right, left for scroll eventType, otherwise undefined
	onElementInView: function(){},

	// Callback for when an element goes out of view. The same data object will
	// be passed in as described in onElementInView.
	onElementOutOfView: function(){},

	// Whether or not to turn on infinite scroll watching. If true, callbacks
	// will be fired when the container reaches the end.
	infiniteScroll: false,

	// Similar to watchOffset, number of pixels to offset what is considered the
	// end of the container.
	infiniteOffset: 0,

	// Callback for when container has reached its vertical end. A data object
	// will be passed into the callback, consisting of the following:
		// eventType: scroll, resize, refresh or scrollwatchinit if the container
		// is at its end on initial load of the page
		// direction: up, down, right, left for scroll eventType, otherwise undefined
	onInfiniteYInView: function(){},

	// Callback for when container has reached its horizontal end. The same data
	// object will be passed in as described in onInfiniteYInView.
	onInfiniteXInView: function(){}
}

Methods

Below are the public methods available on each ScrollWatch instance:

refresh()

Rechecks the dom for new elements to watch. Call this after injecting new content into the DOM that matches an existing watch you have setup.

destroy()

Destroy the instance of ScrollWatch

pauseInfinitScroll()

Pause infinite scroll watching.

resumeInfinitScroll()

Resume infinit scroll watching.
